## About The Project

This project was built as a primary practical submission for Day 1 of the Python Intermediate Internship framework. It models a baseline educational administration tool handling student metrics.

### Key Highlights:
* **Classes & Objects:** Modeled distinct data management matrices for handling academic profiles.
* **Inheritance Layering:** Inherits base structural configurations from a core parent `Member` class into a target child `Student` class.
* **Encapsulation:** Protects internal data records from external structural breaks by routing updates through designated object methods.
